# Collection Page Keyboard Shortcuts

> Owner-only shortcuts. Available when viewing a collection you own.

## Global Shortcuts

These shortcuts work regardless of whether an image is highlighted.

| Shortcut | Action |
|----------|--------|
| `Cmd` + `A` | Enter multi-select mode and select all images |

## Navigation

Use arrow keys to highlight and navigate through images in the grid.

| Shortcut | No Image Highlighted | Image Highlighted |
|----------|---------------------|-------------------|
| `→` | Highlight first image | Move right |
| `←` | Highlight last image | Move left |
| `↑` | — | Move up |
| `↓` | — | Move down |
| `Space` | — | Toggle selection of highlighted image |
| `Enter` | — | Open image modal |
| `Escape` | Exit selection mode / Go back | Clear highlight |

## Image Actions

Available when an image is highlighted.

### Normal Mode

| Shortcut | Action |
|----------|--------|
| `e` | Edit image |
| `r` | Remove image from collection |
| `d` | Download image |
| `f` | View in fullscreen |
| `a` | Add image to another collection |
| `c` | Set as cover |
| `u` | Replace image |
| `t` | Delete image |

### Selection Mode

When images are selected, these shortcuts act on the selection.

| Shortcut | Action |
|----------|--------|
| `r` | Remove all selected images from collection |
| `a` | Add all selected images to another collection |
| `Space` | Toggle selection of highlighted image |

## Collection Actions

Available when no image is highlighted.

| Shortcut | Action |
|----------|--------|
| `A` | Add images to collection |
| `E` | Edit collection |
| `D` | Delete collection |
| `R` | Toggle reorder mode |
| `U` | Upload to collection |
| `M` | Toggle selection mode |

## Notes

- Shortcuts are owner-only (collection owner or admin).
- Arrow key navigation does not wrap at grid edges.
- Grid columns: 3 on screens < 768px, 5 on screens >= 768px.
- Highlighted images show a **pink ring** in normal mode, **blue ring** in selection mode.
- Clicking an image sets the keyboard highlight.
- Modal/dialog overlays block all shortcuts while open.
